## 💡 서비스 개요

**서비스명:** 우리 아이 보상체계로 키우기

**목표:** 주입식 교육 대신 ‘성과-보상 중심의 자율 학습’을 통해 아이의 주체성과 성취감을 길러주는 교육 플랫폼

---

## 🧭 핵심 컨셉

- **모험가(아이)** : 도전을 통해 재화를 얻고 성장하는 존재
- **의뢰인(부모)** : 아이에게 과제를 제시하고, 현실 보상을 제공하는 멘토
- **재화(골드, 코인 등)** : 아이의 노력과 성취를 수치로 표현
- **상점(부모 커스텀)** : 재화를 실제 만족감으로 연결시키는 공간
- **테마:** 강아지 용사 🐶(ENFP, 낙천적) × 고양이 의뢰인 🐱(ISTJ, 현실적)

## 🔁 서비스 플로우

### 1️⃣ 의뢰 플로우

1. 부모 → 의뢰 등록 (예: “숙제 끝내기” / 난이도: 중 / 보상: 50코인)
2. 부모는 등록 시 **재화 선결제** (소모)
3. 아이 → 해당 의뢰 수행 후 ‘완료’ 버튼 클릭
4. 부모 → 수행 내용 확인 후 “승인” 시 아이에게 보상 지급
5. 미승인 시 → 다시 ‘미해결 상태’로 회귀

---

### 2️⃣ 보상 플로우

1. 부모 → 상점 아이템 등록 (예: “15분 게임시간”, “피자데이”)
2. 아이 → 상점에서 재화로 구매
3. 부모 → 실제 보상 이행 (약속 기반)

---

### 3️⃣ 재화 플로우

- **기본 지급:** 매일 100코인 자동 충전
- **의뢰 등록 비용:** 최소 25코인 (25, 50, 100 단위)
- **부족 시 결제 유도:** 실제 화폐로 코인 충전
- **교육 메시지:** “의뢰 난이도에 맞는 보상 책정이 아이 성장에 도움됩니다.”

## 🎯 MVP 목표 (최소기능제품)

1. 1:1 관계만 지원 (가족 1쌍 중심)
2. **Daily 의뢰만 가능** (장기 의뢰는 이후 버전에서)
3. 간단한 캐릭터 일러스트 or 프로필 아이콘 형태
4. UI는 **아이 친화적 / 부모는 관리 중심형**
5. 로그인, 회원가입, JWT 인증 (역할 기반 분리)

| 구분 | 기술 선택안 |

| **Frontend** | React (또는 React Native로 앱 확장성 고려), Tailwind / MUI |
