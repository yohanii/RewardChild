제목 : '우리 아이 보상체계로 키우기'

목적 : 아이를 보상체계로 교육함으로써, 아이로하여금 성취감을 느끼게 하고, 주체성을 가지게 교육한다.
한국에선 주입식 교육이 강하기 때문에, 이런 교육체계로 교육하는게 더 건강하다고 생각하기 때문.

설명 :

1. 부모님이 의뢰를 걸고, 그에 맞는 재화를 건다.
2. 아이는 의뢰를 해결하고, 재화를 얻는다.
3. 아이는 상점에서 재화를 통해 보상을 구매할 수 있음.
4. 상점주인은 부모님이고, 보상과 구매할 때 드는 소모 재화를 커스텀할 수 있음.
5. 부모님은 아이가 구매한 보상을 현실에서 들어주기로 약속.

BA:
매일 100개 화폐정도를 지급해줄 건데,
부모님이 의뢰를 하기에 부족하다면, 실제 돈을 내고 재화를 구매해야함.
그래서 의뢰 최소비용을 도입할 것임.
25단위이며, Daily 의뢰는 25, 50, 100만 가능.
이런식으로 구매를 유도함.
의뢰의 난이도에 따라 보상으로 받을 재화를 알맞게 설정하는것이 교육에 효과적이라는 메세지도 보여줄것임.

컨셉:
동물 모험가
아이(모험가) : 강아지(리트리버), 용사, 명랑, 꿈과 희망, ENFP
부모님(의뢰인, 상점주인) : 고양이(회색 아메리칸숏헤어), 계산적, 현실적, ISTJ

MVP목표:

1. 1-1관계 (다대다는 추후 예정)
2. 캐릭터 디자인 (간단하게라도 넣는게 나을지 고민)
3. Daily의뢰만 (2일이상 의뢰는 추후 예정)

화면:
1. 모험가
(1) 의뢰게시판 화면 - 의뢰 완료할 수 있음.
(2) 상점 구매 화면 - 모은 재화로 보상을 구매할 수 있음.
2. 의뢰인
(1) 의뢰게시판 화면 - 의뢰 등록할 수 있음.
(2) 상점 물품 등록 화면
(3) 재화 구매 화면


플로우:
1. 의뢰인 의뢰 등록(재화소모) -> 모험가 의뢰 해결 -> 의뢰인 확인(부모님이 승인 안하면, 다시 미해결처리로 바꿔야함) -> 모험가 재화획득
2. 의뢰인 상점 상품 등록 -> 모험가 상품 구매(재화소모) -> 부모님 현실에서 보상 이행
